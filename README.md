## Installation
#### Program is downloaded form git repo then opened with VS Code or similiar. Navigate to the file index.js and right click and open in intergrated terminal. Once bash is open then run npm init and npm i express, npm i mysql2, npm i console.table --save. This will add the core functionality to the program. Then in bash execute command node index.js and follow promts.

## Usage
#### Program is used by operating from node js in bash terminal and perfomring operations within the program in the terminal. Currently used to look at and review employee roster.
